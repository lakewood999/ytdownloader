Welcome to ytdownloader! This is an open-source web application that allows you to download videos and/or extract audio from YouTube videos. The source code to this website is open-sourced at Github and Gitlab. Developed and maintained by Steven Su.

Download YouTube Videos

Instructions

  1. Copy and paste your YouTube URL into the box above. (To get the URL on the mobile YouTube app, click on Share -> Copy Link and return to this website)
  2. Choose whether you want video or audio-only by using the dropdown next to the URL box.
  3. Choose other download options such as quality and encoding. Note: this is currently not implemented. Options for video/audio and quality will be added later.
  4. Press the "Download" button to start the download process. This will start a download request in the server processing chain and as soon as resources are available, your YouTube video will be downloaded, processed, and then sent to this client. Please note that depending on the server load, you may have to wait for some time. A status window will appear below the form so that you can check your progress.
  5. Note: The download percentage may loop around after reaching 100% if you choose the "Audio + Video" option
