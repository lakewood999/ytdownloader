Welcome to ytdownload! This is an open-source web application that allows you to download videos and/or extract audio from YouTube videos. The source code to this website will be made available soon. Developed and maintained by Steven Su.

Download YouTube Videos

Instructions

  1. Copy and paste your YouTube URL into the box above. (To get the URL on the mobile YouTube app, click on Share -> Copy Link and return to this website)
  2. Choose your download options. Note: this is currently not implemented. Options for video/audio and quality will be added later.
  3. Press the "Download" button to start the download process. This will start a download request in the server processing chain and as soon as resources are available, your YouTube video will be downloaded, processed, and then sent to this client. Please note that depending on the server load, you may have to wait for some time. A status window will appear below the form so that you can check your progress.
